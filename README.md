本仓库主要用于收集[华中科技大学开源镜像站](https://mirrors.hust.edu.cn)用户增删改镜像需求，方便，快捷的管理和追踪需求。

## 反馈问题

如您在使用[华中科技大学开源镜像站](https://mirrors.hust.edu.cn)的过程中遇到问题，欢迎您随时点击[这里](https://github.com/hust-open-atom-club/mirrorrequest/issues/new/choose)进行反馈。

## 使用帮助

[点击查看](https://mirrors.hust.edu.cn/docs)

## 查看Tunasync配置

本仓库 `tunasync-config` 目录中存放镜像站 tunasync 相关配置文件，在 `tunasync-scripts` 目录中存放同步镜像所用到的脚本。
